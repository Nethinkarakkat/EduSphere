## Getting Started

### Account Registration

1. Navigate to the EduSphere login page
2. Click **"Sign Up"**
3. Fill in your information:
   - **Name:** Full name
   - **Email:** Institutional email address
   - **Password:** Strong password (minimum 6 characters)
   - **Confirm Password:** Re-enter password
   - **Role:** Select "Faculty"
4. Click **"Sign Up"**

### Account Approval

Faculty accounts require admin approval before access is granted:

1. After registration, you'll see a pending approval message
2. Wait for admin approval (usually within 24 hours)
3. You'll receive an email when your account is approved
4. Log in with your credentials

### First-Time Setup

1. Complete your profile:
   - Add your designation/title
   - Specify your subject specialization
   - Add your faculty ID (if applicable)
2. Upload a profile picture (optional)
3. Change your password if needed

## Classroom Management

### Creating a Classroom

1. Navigate to **Classrooms** from the dashboard
2. Click **"Create Classroom"**
3. Fill in the required information:
   - **Classroom Name:** e.g., "Physics 101 - Section A"
   - **Subject:** e.g., "Physics"
   - **Description:** Optional description of the classroom
4. Click **"Create Classroom"**

**Note:** A unique classroom code will be automatically generated. Share this code with students for enrollment.

### Viewing Classrooms

1. Navigate to **Classrooms**
2. View all your classrooms including:
   - Classroom name and subject
   - Number of enrolled students
   - Number of associated exams
   - Creation date

### Editing a Classroom

1. Navigate to **Classrooms**
2. Click on the classroom you want to edit
3. Click **"Edit Classroom"**
4. Update the required fields
5. Click **"Update Classroom"**

### Deleting a Classroom

1. Navigate to **Classrooms**
2. Click on the classroom you want to delete
3. Click **"Delete Classroom"**
4. Confirm the deletion

⚠️ **Warning:** Deleting a classroom will:
- Remove all enrolled students
- Remove all associated exams
- Delete all exam submissions
- This action cannot be undone

### Viewing Classroom Details

Click on any classroom to view:
- Classroom information
- Enrolled students
- Associated exams
- Student submissions

### Sharing Classroom Code

Share the classroom code with students:

1. Navigate to **Classrooms**
2. Click on the desired classroom
3. Copy the classroom code
4. Share it with students via email, LMS, or in class

Students use this code to join the classroom.

## Exam Creation

### Creating an Exam

1. Navigate to **Exams** from the dashboard
2. Click **"Create Exam"**
3. Fill in the required information:
   - **Title:** Exam title (e.g., "Midterm Exam")
   - **Subject:** Subject of the exam
   - **Exam Date:** Date and time of the exam
   - **Duration:** Duration in minutes
   - **Pass Percentage:** Minimum percentage to pass (e.g., 50)
   - **Classroom:** Select the classroom (optional)
   - **Instructions:** Exam instructions for students
4. Click **"Create Exam"**

### Editing an Exam

1. Navigate to **Exams**
2. Click on the exam you want to edit
3. Click **"Edit Exam"**
4. Update the required fields
5. Click **"Update Exam"**

⚠️ **Note:** You can only edit exams that are in "Draft" status. Launched exams cannot be edited.

### Deleting an Exam

1. Navigate to **Exams**
2. Click on the exam you want to delete
3. Click **"Delete Exam"**
4. Confirm the deletion

⚠️ **Warning:** Deleting an exam will:
- Remove all questions
- Delete all student submissions
- This action cannot be undone

### Previewing an Exam

Before launching, preview the exam:

1. Navigate to **Exams**
2. Click on the exam you want to preview
3. Click **"Preview Exam"**
4. Review the exam as students will see it

### Moving Exams Between Classrooms

1. Navigate to **Classrooms**
2. Click on the source classroom
3. Find the exam you want to move
4. Select the target classroom
5. Click **"Move Exam"**

### Copying Exams to Another Classroom

1. Navigate to **Classrooms**
2. Click on the source classroom
3. Find the exam you want to copy
4. Select the target classroom
5. Click **"Copy Exam"**

**Note:** Copying creates a new draft exam in the target classroom. Submissions are not copied.

## Question Management

### Adding Questions to an Exam

1. Navigate to **Exams**
2. Click on the exam you want to add questions to
3. Click **"Add Questions"**
4. Fill in the question details:
   - **Question:** The question text
   - **Option 1-4:** Four answer options
   - **Correct Answer:** Select the correct option (1-4)
   - **Difficulty:** Easy, Medium, or Hard
   - **Marks:** Points for this question
5. Click **"Add Question"**
6. Repeat for additional questions

### Importing Questions from Question Bank

1. Navigate to **Exams**
2. Click on the exam
3.Click **"Add Questions"**
4. Click **"Import from Question Bank"**
5. Select questions from your question bank
6. Click **"Add Selected Questions"**

### Viewing Exam Questions

1. Navigate to **Exams**
2. Click on the exam
3. Click **"View Questions"**
4. Review all questions in the exam

### Editing Exam Questions

1. Navigate to **Exams**
2. Click on the exam
3. Click **"Edit Questions"**
4. Update question details as needed
5. Click **"Update Questions"**

### Deleting Questions

1. Navigate to **Exams**
2. Click on the exam
3. Click **"Edit Questions"**
4. Click **"Delete"** next to the question
5. Confirm the deletion

## Question Bank

The Question Bank allows you to create reusable questions for multiple exams.

### Adding Questions to the Bank

1. Navigate to **Question Bank** from the dashboard
2. Click **"Add to Question Bank"**
3. Fill in the question details:
   - **Question:** The question text
   - **Option 1-4:** Four answer options
   - **Correct Answer:** Select the correct option (1-4)
   - **Category:** Question category (e.g., "Physics", "Chemistry")
   - **Difficulty:** Easy, Medium, or Hard
4. Click **"Add to Bank"**

### Viewing Question Bank

1. Navigate to **Question Bank**
2. View all your banked questions including:
   - Question text
   - Category
   - Difficulty
   - Creation date

### Filtering Question Bank

Use filters to find specific questions:
- **Search:** Search by question text
- **Category:** Filter by category

### Editing Banked Questions

1. Navigate to **Question Bank**
2. Click on the question you want to edit
3. Update the question details
4. Click **"Update Question"**

### Deleting Banked Questions

1. Navigate to **Question Bank**
2. Click on the question you want to delete
3. Click **"Delete Question"**
4. Confirm the deletion

## Exam Launching

### Launching an Exam

Make an exam available to students:

1. Navigate to **Exams**
2. Click on the exam you want to launch
3. Click **"Launch Exam"**
4. Confirm the launch

**Note:** Once launched, students can attempt the exam. You cannot edit a launched exam.

### Exam Status

- **Draft:** Exam is being created, not visible to students
- **Launched:** Exam is live, students can attempt it
- **Published:** Results have been published

## Results and Grading

### Viewing Student Results

1. Navigate to **Results** from the dashboard
2. Use filters to find specific results:
   - **Subject:** Filter by subject
   - **Classroom:** Filter by classroom
   - **Exam:** Filter by exam
   - **Date Range:** Filter by submission date
   - **Result Status:** Filter by pass/fail
   - **Search:** Search by student name
3. View results including:
   - Student name
   - Exam title
   - Score and percentage
   - Pass/Fail status
   - Submission date

### Publishing Results

Make results visible to students:

1. Navigate to **Results**
2. Find the exam you want to publish results for
3. Click **"Publish Results"**
4. Confirm the publication

**Note:** Students can only view results after they are published.

### Viewing Individual Submissions

1. Navigate to **Results**
2. Click on a student's result
3. View detailed submission including:
   - Student answers
   - Correct answers
   - Score breakdown
   - Tab switch count (integrity check)

### Exporting Results

#### CSV Export

1. Navigate to **Results**
2. Apply desired filters
3. Click **"Export to CSV"**
4. The file will download automatically

#### PDF Export

1. Navigate to **Results**
2. Apply desired filters
3. Click **"Export to PDF"**
4. The file will download automatically

## Best Practices

### Exam Creation

1. **Clear Instructions:** Provide clear exam instructions
2. **Appropriate Difficulty:** Balance question difficulty
3. **Time Management:** Set realistic duration based on question count
4. **Preview:** Always preview exams before launching
5. **Backup Questions:** Maintain a question bank for reuse

### Question Quality

1. **Clear Wording:** Use clear, unambiguous language
2. **Single Correct Answer:** Ensure only one correct option
3. **Relevant Content:** Align questions with learning objectives
4. **Variety:** Include different difficulty levels
5. **Review:** Review questions for errors before adding

### Classroom Management

1. **Clear Communication:** Share classroom codes clearly
2. **Regular Updates:** Keep classroom information current
3. **Student Engagement:** Monitor student participation
4. **Timely Feedback:** Publish results promptly

### Results and Analytics

1. **Regular Review:** Regularly review analytics
2. **Identify Trends:** Look for performance patterns
3. **Provide Feedback:** Use results to guide teaching
4. **Data-Driven Decisions:** Make informed decisions based on data

## Troubleshooting

### Common Issues

#### Students Cannot Join Classroom

- Verify classroom code is correct
- Check if classroom is archived
- Ensure student is using the correct code format

#### Exam Not Visible to Students

- Verify exam is launched (not in draft)
- Check if exam is assigned to the correct classroom
- Ensure exam date has arrived

#### Questions Not Saving

- Check all required fields are filled
- Verify correct answer is selected
- Ensure marks are specified

#### Results Not Publishing

- Verify all students have submitted
- Check if results are already published
- Ensure you have permission to publish
